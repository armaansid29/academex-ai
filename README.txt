# Acadex

Acadex is a sleek, AI-powered professor discovery tool. It lets users explore university faculty profiles, view research summaries, and interactively learn more about a professor’s work by clicking on their profile to generate follow-up AI insights.  

The design is black, minimal, and futuristic — with smooth animations and an intuitive card-based layout.  

## Features
- **AI-powered summaries** — Condensed professor bios in 3 sentences or less.  
- **Research keywords** — Quickly understand areas of expertise.  
- **Interactive follow-ups** — Click a professor’s card to ask the AI for deeper insights.  
- **Clean, modern UI** — Animated cards with a sleek dark theme.  

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript (ES Modules)
- **API:** OpenAI for AI-generated content
- **Styling:** Custom CSS animations + Google Fonts

## Getting Started
1. Clone the repo:  
   ```bash
   git clone https://github.com/yourusername/acadex.git
   cd acadex
   ```
2. Install dependencies (if any) and set your OpenAI API key in `.env`.  
3. Open `index.html` in your browser to start exploring.
